Đăng nhập vào Facebook Developer Portal và tạo một ứng dụng mới.

Thêm sản phẩm Messenger và cấu hình kết nối với Facebook Page của bạn.

Trong phần cài đặt Messenger, nhập URL webhook (ví dụ: https://your-domain.com/webhook) và verify token trùng với giá trị FB_VERIFY_TOKEN bạn đã cấu hình.

Chọn các sự kiện cần nhận (ví dụ: tin nhắn gửi đến).
